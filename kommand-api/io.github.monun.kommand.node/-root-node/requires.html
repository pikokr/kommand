requires

abstract override fun requires(requires: KommandSource.() -> Boolean)

명령 실행에 필요한 권한을 체크합니다.

이 함수는 모든 하위 노드에 영향을 줍니다.

kommand {
  register("mycmd") {
  requires { hasPermission(4) }
    then("first") { // mycmd first
      executes {
      // 4 레벨 권한이 있어야 실행됨
      }
      then("second") { // /mycmd first second
        executes {
        // 4 레벨 권한이 있어야 실행됨
        }
      }
    }
  }
}
